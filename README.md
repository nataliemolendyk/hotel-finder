## What your app does

My app uses the data from the Google API and displays a card for 26 hotels that includes an image, address, price, rating, and room types. The user can click on more details and highlights as well as a descriptions of each room type is given. There are filters at the top for the users to sort by name, airport, min and max price, rating, and allows the user to add hotels to their favorites list.

## Which API you used (with link)

I used the Google Hotels API from SearchAPI (https://www.searchapi.io/docs/google-hotels-api).
